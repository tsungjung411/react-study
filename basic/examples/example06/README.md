## 說明：
- 顯示更複雜的、階層式資訊：
```html
<ul>
  <li>Google</li>
  <li>Facebook</li>
  <li>Microsoft</li>
</ul>
```
<br>

- 使用 JSX 語法來表達（亦即 javascript 裡有 XML 標籤）
- 使用 React 單一元件檔案(.jsx)來產生傳統 JS 檔(.js)
```bash
$ npm install babel-preset-react
$ babel --presets react App.jsx -o App.js # output
```
